# Get the Project

Before you can start coding, you need a copy of this project on your computer. Pick one of the two options below.

&nbsp;

## Option A: Clone with Git (recommended)

This keeps your work connected to GitHub so you can save and share changes later.

1. **Copy the repository URL** — click the green **Code** button on the GitHub page and copy the HTTPS URL (it looks like `https://github.com/username/repo.git`).

2. **Open a terminal** on your computer.

3. **Run the clone command**, replacing the URL with the one you copied:

```bash
git clone https://github.com/your-username/your-repo.git
```

4. **Move into the project folder:**

```bash
cd your-repo
```

5. Open the folder in your IDE and continue to [Getting to Know Your IDE](getting_to_know_your_ide.md).

> **💡 Having trouble?** You can ask your AI assistant to do this for you — just say: **"Clone this repository for me: `<paste the URL here>`"**

&nbsp;

## Option B: Download as ZIP

No Git required — good for a quick start.

1. **Click the green `Code` button** on the GitHub page.
2. Select **`Download ZIP`**.
3. **Unzip** the downloaded file to a folder of your choice.
4. Open that folder in your IDE and continue to [Getting to Know Your IDE](getting_to_know_your_ide.md).

> **💡 Note:** The ZIP option does not include Git history, so you won't be able to push changes back to GitHub without extra steps. Use Option A if you plan to save your work.
